# Unity_SpriteNumber
Unityで数字をスプライトで表示したい時に使えます。<br>

- ゼロを表示するか
- 配置（左揃え、中央揃え、右揃え）
- スプライトサイズ、オフセットの設定
などの設定する事ができます。
